# Pilot Offer: AI-Assisted SDLC Hardening

## Goal

Apply schema-governed agent workflow constraints to one repository or one workflow.

## 30-Day Pilot

1. Audit the current AI or agent workflow.
2. Identify scope drift, token waste, unsafe actions, or source-of-truth gaps.
3. Add bounded role templates and handoff expectations.
4. Add validation and review expectations.
5. Run one or two real tasks through the workflow.
6. Compare before and after results.

## Possible Metrics

- Time from research to ticket.
- Time from ticket to pull request.
- Rework count.
- Review comment count.
- First-pass CI success.
- Out-of-scope file edits.
- Undocumented decisions.
- Token or cost per accepted task.

This pilot framing is exploratory. It does not promise exact savings or guaranteed outcomes.
