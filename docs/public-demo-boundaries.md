# Public Demo Boundaries

This repository is intentionally incomplete.

It is safe for portfolio review, public discussion, and pilot conversations because it avoids implementation-grade details.

## Included

- High-level architecture language.
- Governance principles.
- Non-executable toy examples.
- A pilot discussion outline.
- A proprietary notice.

## Excluded

- Full private framework.
- Complete schema contracts.
- Validation machinery.
- Role generation details.
- CLI direction.
- Runtime integration patterns.
- Reusable orchestration logic.

The examples in this repository are toy examples only. They are not compatible with, or sufficient to reproduce, the private system.
